Flashcard-Learning-Tool

This tool helps users learn by showing flashcards with a question on the front and the answer on the back.
You can "flip" the cards to see the answers and use the "Next" and "Previous" buttons to move through the set.

Password-Generator

This tool generates secure passwords with customizable options.

Preview-Music

This tool allows users to search for music using the Spotify API and play 30-second song previews.

bg-Changer

Just a simple background-color changer app. It provides the idea about how to apply React Hooks and State management.

To-Do App

This tool help users to manage their tasks by providing a user-friendly interface for adding, editing, and deleting tasks.
The app also supports task filtering based on status (All, Pending, Completed).

Shortest_Path_Visualizer

This is a project for visualizing the shortest path from source to destination in graph using Dijkstra algorithm. The current moves used in the application are based on chess moves (rook, bishop, and queen).

generativeAi
